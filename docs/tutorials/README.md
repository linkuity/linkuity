# Tutorials

Hands-on, step-by-step guides for learning Linkuity. Unlike the reference and architecture docs,
these teach by doing — copy the commands and watch what happens.

| Tutorial | What you'll learn |
|----------|-------------------|
| [Build a durable MDM project from scratch with the CLI](cli-durable-mdm-quickstart.md) | Use the command-line tool end to end: create a durable project, load two sources, see automatic merges and the human-review queue, inspect golden records and their history, and optionally view the result as a Neo4j graph. Beginner-friendly; only needs the .NET SDK. |
| [Build a durable product catalog MDM project with the CLI](cli-durable-product-catalog.md) | Use the command-line tool with a product taxonomy: load catalog, supplier, and marketplace feeds; match products by SKU and GTIN; see source-priority merge rules choose trusted fields; inspect a review task for a lookalike product; and view golden product history. Beginner-friendly; only needs the .NET SDK. |

New to Linkuity? Start with the tutorial above, then explore the worked
[`samples/`](../../samples/) and the [architecture overview](../architecture.md).
